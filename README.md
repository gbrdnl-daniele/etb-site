# ETB Cinematic Experience — Developer Edition

Questa versione parte dal progetto ETB inviato dall'utente e lo riorganizza in una struttura più leggibile e mantenibile.

## Avvio

Apri la cartella con Visual Studio Code e usa **Live Server** su `index.html`.

## File più importanti

- `js/timing.js` — tutti i tempi dell'intro e della Hero.
- `js/storyboard.js` — motore della sequenza introduttiva.
- `js/background.js` — canvas con particelle, fumo e fasci di luce.
- `css/storyboard.css` — stile delle frasi iniziali.
- `css/effects.css` — sfondo, glow, fumo, grana e fader nero.
- `css/hero.css` — titoli, Hero, pulsanti e card.

## Font

Questa Developer Edition usa Google Fonts online:

- Barlow Condensed
- Cinzel
- Inter

Per la versione pubblicabile/offline si potranno scaricare i font in formato `woff2` e caricarli con `@font-face`.
